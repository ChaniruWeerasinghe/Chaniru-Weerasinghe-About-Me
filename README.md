# Chaniru Weerasinghe | About Me

A minimal, professional profile page built to showcase my professional links and GitHub account transition.

## Overview

This project is a single-page application (SPA) designed with a focus on minimalism and professional clarity. It provides a central location for my LinkedIn profile and clarifies my transition to a new primary GitHub account.

## Built With

- Vite
- Vanilla JavaScript
- Custom CSS (Mobile-First)

## Local Development

To run this project locally, follow these steps:

1. Clone the repository.
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

## Deployment

This project is configured for automatic deployment to GitHub Pages via GitHub Actions. Any push to the main branch will trigger a new build and deployment.
